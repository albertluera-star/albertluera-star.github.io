# Assets Folder Structure

Place your media files according to this structure:

## Landing Page
```
assets/
├── landing-video.mp4     (or .mov)
└── profile.jpg           (for About page)
```

## Projects
```
assets/projects/
├── project1/
│   ├── cover.jpg
│   ├── image1.jpg
│   ├── image2.jpg
│   └── video.mp4 (optional)
├── project2/
│   └── ...
└── project3/
    └── ...
```

## Personal Art
```
assets/personal/
├── artwork1.jpg
├── artwork2.jpg
├── artwork3.jpg
├── video1.mp4
└── ...
```

## Quick Start Guide

1. Create the folder structure shown above
2. Add your video file for the landing page
3. Organize your project files into separate folders
4. Add your personal artwork files
5. Update the HTML files with your actual filenames

## Image Specifications

- **Landing Video**: 1920x1080px, H.264, under 10MB
- **Project Covers**: 1200x900px minimum
- **Personal Art**: 1200px width minimum
- **Profile Photo**: 800x1000px (portrait)

## File Naming

Use lowercase and hyphens:
- ✅ `project-name.jpg`
- ✅ `personal-artwork-01.jpg`
- ❌ `Project Name.jpg`
- ❌ `personal artwork 01.jpg`
